# akire

## Description

[akire7's](https://github.com/akire7) personal website.

## Getting Started
1. Install dependencies
```
pnpm install
```
2. Run akire locally
```
pnpm dev
```

## Technology

Frameworks:
- Next.js

Database:
- MongoDB (NoSQL)

Deployment:
- Vercel

Version Control:
- GitHub
- Git
